# 简报模板

## 早间简报（06:00 生成）

**时间范围**: 昨日 18:00 ～ 今日 06:00

```markdown
## 🌅 早间简报 | {日期}

### 📋 昨日回顾
- **工作时间**: 昨日 18:00 ～ 今日 06:00
- **完成事项**:
  - {事项 1}
  - {事项 2}

### 🔄 进行中
- {进行中事项}

### ⚠️ 问题与风险
- {问题描述}（如有）

### 📅 今日计划
- {今日计划事项}
```

---

## 晚间简报（18:00 生成）

**时间范围**: 今日 06:00 ～ 18:00

```markdown
## 🌆 晚间简报 | {日期}

### 📋 今日工作
- **工作时间**: 今日 06:00 ～ 18:00
- **完成事项**:
  - {事项 1}
  - {事项 2}

### 🔄 进行中
- {进行中事项}

### ⚠️ 问题与风险
- {问题描述}（如有）

### 📅 明日计划
- {明日计划事项}
```

---

## 生成规则

1. **时间范围**:
   - 早间：昨日 18:00 ～ 今日 06:00（12 小时）
   - 晚间：今日 06:00 ～ 18:00（12 小时）

2. **内容来源**:
   - Git 提交记录（COMMIT_LOG.md）
   - 会话历史
   - 文件变更记录

3. **必须标注**:
   - ⚠️ 问题风险必须明确标注
   - 如无风险，写"无重大风险"

4. **发送目标**:
   - 接收人：pm（产品经理）
   - 会话：agent:pm:main

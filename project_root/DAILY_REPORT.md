# 每日汇报机制

## 汇报时间

- **早汇报**: 北京时间 07:00（汇报前日 19:00 ~ 今日 07:00）
- **晚汇报**: 北京时间 19:00（汇报今日 07:00 ~ 19:00）

## 汇报流程

```
各 agent 生成简报 → 发送给用户 + @document + @pm → @document 整理 → @Notion归档 归档
                                    ↑
                              @pm 审核
```
